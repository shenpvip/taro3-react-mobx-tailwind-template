# taro3-react-mobx-tailwind-template

基于 Taro3 的 React 多端项目模版，默认支持编译成微信小程序，其他小程序自行测试

## 技术栈

- Taro
- React Hooks
- Mobx
- NutUI

## 运行环境

- Node >=16.14.2
- Pnpm 8+

## 支持特性

- 🏠 基于 Taro3.6.23
- 📦 支持 React
- 🐑 CSS 预处理器( SCSS )
- 🎨 TailwindCSS - 世界上最流行，生态最好的原子化 CSS 框架
- 🥣 完全使用 TypeScript 开发
- 🔛 request 拦截器封装
- 🔥 单元测试(jest+enzyme)
- 🧬 iconfont 支持(替换为自己的 iconfont 文件)
- 🌩️ 使用多核心及缓存提升编译速度
- 💰 更多特性持续迭代中...
